RawPage
=======

My personal website. It's like a Terminal.

Commands
========

* **help**: list all disponible commands.
* **ls**: list all pages.
* **cat**: show/open pages.
* **hostname**: show site address.
* **clear**: clear screen.
* **date**: display current date.
* **echo**: display a text.
* **history**: print command history.
